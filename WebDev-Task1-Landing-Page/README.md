# Task 1 – Landing Page

A modern and responsive landing page developed as part of the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**. The project showcases a clean user interface, responsive design, smooth animations, and interactive elements to deliver an engaging user experience.

## Features

- Responsive design for all screen sizes
- Modern and minimal UI
- Light/Dark mode toggle
- Smooth scrolling and scroll reveal animations
- Interactive navigation bar
- Hero, Features, Testimonials, and CTA sections
- Clean and semantic HTML structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes

- Responsive Web Design
- CSS Flexbox & Grid
- DOM Manipulation
- Theme Switching
- UI/UX Design Principles
- Frontend Development Best Practices
