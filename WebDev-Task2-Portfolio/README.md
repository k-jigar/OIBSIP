# Task 2 – Portfolio Website

A modern, responsive personal portfolio website developed as part of the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**. The website showcases personal information, technical skills, projects, education, and contact details through a clean, interactive, and user-friendly interface.

## Features

- Fully responsive design
- Modern and minimal UI
- Interactive navigation
- About, Skills, Projects, Education, and Contact sections
- Smooth scrolling and scroll reveal animations
- Contact form
- Clean and semantic HTML structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes

- Responsive Web Design
- CSS Flexbox & Grid
- JavaScript DOM Manipulation
- Interactive UI Development
- Portfolio Design Principles
- Frontend Development Best Practices
