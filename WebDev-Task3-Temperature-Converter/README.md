# Task 3 – Temperature Converter

A responsive and interactive temperature converter developed as part of the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**. The application converts temperatures between Celsius, Fahrenheit, and Kelvin instantly with real-time updates and input validation.

## Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Real-time temperature conversion
- Input validation with error handling
- Displays conversion formula
- Responsive and modern user interface
- Unit swap functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Learning Outcomes

- JavaScript DOM Manipulation
- Event Handling
- Form Validation
- Mathematical Calculations
- Responsive UI Design
- Frontend Development Best Practices
